# How to use existing components

## Window
```yaml
title: "r-type"
width: 1900
height: 1080
max_fps: 60
```

# Sprite
```yaml
texture: "path_to_texture"
scale_x: 1
scale_y: 1
```

# Transform
```yaml

width: 200
height: 200

```
